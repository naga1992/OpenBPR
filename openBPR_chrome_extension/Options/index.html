Adding Cognizant Intelligent Test Scripter Certificate to browsers

Chrome Web browser
  • From the location of the installation file, open the Extensions folder
  • Launch the Ext_Certificate.cert file. The Certificate window is displayed
  • To install the Certificate, perform the following steps in the Certificate window that is displayed:

    Click the button
    Click the Next button and select Place all certificates in the following store.
    Browse and select Trusted Root Certification Authorities
    Click the Next button followed by Finish.

Mozilla Firefox Web browser
  • Launch Cognizant Intelligent Test Scripter application
  • Launch one of the following options(Mandatory):

    Object Spy
    Object Heal
    Recorder

  • Launch Firefox browser
  • Navigate to Options > Advanced > Certificates
  • Select the option Select one automatically
  • Click View Certificate, navigate to Servers and click Add exception. The Add Security Exception window is displayed
  • In the Location field of the Add Security Exception window, type the URL: https://localhost:8887
  • Click Get Certificate and Confirm Security Exception
  • In case of any challenges in adding the security certificate, ensure to perform the following steps:

    Open Internet Options
    Navigate to Connections>Lan Settings>Advanced
    In the exceptions area add localhost
    Save the settings and again try to add certificate
